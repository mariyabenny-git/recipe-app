# 🍳 Eco-Chef AI – Smart Recipe Platform

A mobile-friendly, feature-rich recipe application designed for a seamless and accessible cooking experience. This project showcases structured state management, dynamic UI rendering, and a heavy focus on clean user experience.

## ✨ Key Features

* **Interactive "Cook Mode":** A dedicated, hands-free UI display mode featuring large, readable text so you can follow steps easily while cooking.
* **Smart UI/UX:** A beautiful, eye-friendly Dark Theme combined with vibrant, modern accent colors.
* **Fully Responsive:** Optimized meticulously for mobile viewports, ensuring a premium experience on any screen size.
* **Component-Based Logic:** Structured dynamically to allow easy expansion of categories and recipe steps.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Architecture:** Mobile-first, component-driven design
